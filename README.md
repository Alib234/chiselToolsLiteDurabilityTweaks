# Chisel Tools Lite Durability Tweaks

[vintagestory mods page for this](https://mods.vintagestory.at/show/mod/46511)

Increases durability of [Chisel Tools Lite](https://mods.vintagestory.at/chiseltoolslite) tools by 10x

meant to be used together with [Chisel Durability Tweak](https://mods.vintagestory.at/show/mod/32219) which increases the durability of the vanilla chisel by 10x
this mod is a simple patch job so unless Chisel Tools Lite breaks something or something else happens it should always work
